## The Applitools Test Cloud

The Applitools Test Cloud is made up of several distinct modules that can be used together.

 ## Quickstart
Choose your favorite framework or language and get started by adding the Applitools SDK.

### Web SDKs

<div>
  <div class="row is-multiline">
    {webframeworks.map((x) => (
      <div key={x.name} class="col col--3">
        <Link class="card" to={x.href}>
          <div class="card__body" style={{ display: 'flex', gap: 20, alignItems: 'center' }}>
            {x.logo && !x.themed ? (
              <x.logo width="20" alt={x.name} style={{ display: 'block', maxHeight: 20 }} />
            ) : (
              <ThemedImage
                style={{ display: 'block', maxHeight: 20 }}
                alt={x.name}
                width="20"
                sources={{
                  light: useBaseUrl(x.logo.light),
                  dark: useBaseUrl(x.logo.dark),
                }}
              />
            )}
            <p>{x.name}</p>
          </div>
        </Link>
      </div>
    ))}
  </div>
</div>
